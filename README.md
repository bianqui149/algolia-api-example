# Algolia Api example

Algolia library Api.

## Installation

Install via composer or clone the repo from https://github.com/algolia/algoliasearch-client-php.

```bash
composer require algolia/algoliasearch-client-php
```

## Usage

```python
require_once('/vendor/autoload.php');
```
